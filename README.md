# World Cup Simulator

A Python project that simulates World Cup matches, group standings, and eventually knockout rounds.

## Current Features

- Simulates one match between two teams
- Uses basic team ratings
- Adds randomness to make results different each run

## Planned Features

- Group stage simulation
- Points table and standings
- Goal difference tracking
- Round of 32 knockout bracket
- CSV-based team loading
- Basic prediction model using historical data

## How to Run

```bash
python main.py
